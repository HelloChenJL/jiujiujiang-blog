# 助力前端开发的浏览器插件 VisBug

# 前言

VisBug 是一款由 Adam Argyle 所创作的用于 Web 开发的浏览器扩展，它可以在 Google Chrome 、Firefox、Edge 等浏览器中使用。VisBug 的主要目的是为 Web 开发者提供一种更加直观、易用和有趣的方式来开发和设计网站。VisBug 与浏览器的开发者工具类似，但提供了更多的可视化工具，可以在页面上实时编辑 CSS 样式、HTML 元素和布局，以及检查和改进页面的可访问性和性能。

[GitHub地址](https://github.com/GoogleChromeLabs/ProjectVisBug?spm=a2c6h.12873639.article-detail.7.16fc4144CXAhFw)
