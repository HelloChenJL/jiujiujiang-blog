# 一些常用的正则表达式

| 0-100的整数（包括0或100） | /^(?:\[1-9\]\[0-9\]?\|100\|0)$/ |
| --- | --- |
| 0到100之间的数字，包括小数点后最多两位的数字 | /^(\[1-9\]\d?(\.\d{1,2})?\|0\.\d{1,2}\|100(\.\[0\]\*)?)$/ |
| 必填项 | /.+/ |
| 是否包含空格 | /\s/g |
| 是否以空格开头或结尾 | /^\s\|\s$/ |
| 邮箱格式 | /^\w+(\[-+.\]\w+)\*@\w+(\[-.\]\w+)\*\.\w+(\[-.\]\w+)\*$/ |
| 电话格式 | /^(\d{3}-\|\d{4}-)?(\d{8}\|\d{7})$/ |
| 手机号格式 | /^1\d{10}$/ |
| 密码格式（8-15位，且必须包含字母大小写、数字、符号 !@#$%^&\*().） | /^(?=.\*\[a-z\])(?=.\*\[A-Z\])(?=.\*\d)(?=.\*\[!@#$%^&\*.()\])\[a-zA-Z\d!@#$%^&\*.()\]{8,15}$/ |
| 身份证号 | /^(\d{15}$\|^\d{18}$\|^\d{17}(\d\|X\|x))$/ |
| 邮编格式 | /^\[0-9\]{6}$/ |
| 网址URL | /^(http\[s\]:\/\/)?\[A-Za-z0-9\]+\.\[A-Za-z0-9\]+\[\/=\?%\-&\_~\`@\[\\]\':+!\]\*(\[^<>\"\"\])\*$/ |
| 小数 | /^\d+(\.\d+)?$/ |
| 英文 | /^\[A-Za-z\]+$/ |
| 中文 | /^\[\u0391-\uFFE5\]+$/ |
| 中英文 | /^\[A-Za-z\u4e00-\u9fa5\_\]+$/ |
| 中文、数字和字母 | /^\[A-Za-z0-9\u4e00-\u9fa5\]+$/ |
| 中文和字母 | /^\[A-Za-z0-9\u4e00-\u9fa5\_\]+$/ |
| 字母和数字 | /^\[A-Za-z0-9\_\]+$/ |
| 正整数 | /^\[1-9\]\d\*$/ |
| 整数 | /^-?\[1-9\]\d\*$/ |
| 文件路径格式 | /^(\[a-zA-Z\]:)?(((\[\\\/\](?! )\[^:\*?<>\""\|\\\/\]+)+\\?)\|(\\)?)\s\*$/ |
| 16位字符数组 | /^\[A-Za-z\d\]{1,16}$/ |
| 6-15位数字和字母的组合 | /^(?!\[0-9\!@#\$%\^&\\*\(\)\.\]+$)(?!\[a-zA-Z\!@#\$%\^&\\*\(\)\.\]+$)\[0-9A-Za-z\!@#\$%\^&\\*\(\)\.\]{6,15}$/ |
| 最多2位小数的数字 | /^\d+(.\d{1,2})?$/ |
| 最多9位整数3位小数的数字 | /^\[0-9\]\d{0,8}(\.\d{1,3})?$/ |
| 最多有8位整数2位小数的数字 | /^\[0-9\]\d{0,7}(\.\d{1,2})?$/ |
